What is it ?

The-Social-Network is a project made by a college student to develop a clone of social media sites like Facebook and Instagram with core functionalities.

Why is it ?

The motivation behind building this project lies in the following points.

Learn the ins and outs of programming.

Understand the backend structure of such social media platforms.

Learn full stack development.

Improve project portfolio.
